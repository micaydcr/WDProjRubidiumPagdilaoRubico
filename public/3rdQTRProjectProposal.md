# Q3 Proposal Update

## MiCayke Decorating!

## Features 
- Works on both phone and laptop
- Allows the user to download the cake they made
- Allow the user to decide what color the base of their cake will be

## Details
- We will use media queries and flexible layouts to ensure that the buttons, form, and most especially the cake preview can be seen and used properly on different screen sizes
- To allow the users save the cake they made, we will convert the cake preview into an image that they can download. In addition to that, we will also save the users cake design into local storage so that they won't lose any progress whenever they refresh or exit the page
- We will use a combination of html and js to allow users to select the color in which will change the tier color of the cake

## Definition of Done
- Our "definition of done" includes for the features said above to be executed in a manner that is within our capabilities, while also functioning without any issues in our devices, this includes not only laptops, but also mobile phones. To expound,  our definition of done  requires that the features are fully functional, responsive across different screen sizes, and provide a smooth and consistent user experience regardless of the device used.
